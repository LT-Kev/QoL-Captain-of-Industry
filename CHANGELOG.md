# Changelog

All notable changes to `QoL Captain of Industry` will be documented in this file.

The format is inspired by `Keep a Changelog`.

## [0.1.1] - 2026-04-06

### Fixed

- Preserved selected storage and selected mine tower scope while using the `F8` GUI, so selected actions continue to work after clicking inside the mod window

## [0.1.0] - 2026-04-05

### Added

- Rebuilt in-game `F8` QoL window for `Captain of Industry` `Update 4`
- Tabs for `General`, `Vehicles`, `Terrain`, `Storage`, and `Weather`
- Extended game speed controls beyond vanilla `x3`, including `x5`, `x10`, and `x15`
- Vehicle limit controls tied to the live `Update 4` vehicle manager
- Storage tools, including global storage modes and a per-storage inspector button
- Terrain tools for instant mining, instant dumping, terrain conversion, tree helpers, and reserve refills
- Weather tools and practical sandbox helpers for research, workers, unity, maintenance, power, and computing
- Direct repo download artifacts for both the compiled `DLL` and packaged `ZIP`

### Changed

- Updated repository metadata, README, license, and download links for public release
- Switched the public console command prefix to `coi_qol_*`

### Notes

- This first public release focuses on stable `Update 4` compatibility and practical tools
- Older brittle terrain hooks were intentionally left out for now to keep the mod reliable
