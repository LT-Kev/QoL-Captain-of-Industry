QoL Captain of Industry 0.1.1 for Captain of Industry Update 4

This mod adds a rebuilt in-game GUI and matching console commands for practical sandbox and quality-of-life controls.

Install:
1. Build the project or use a packaged release.
2. Enable modding in Captain of Industry.
3. Enable the mod in your save or when creating a new game.

Main usage:
- Press F8 to open the QoL window.
- Or open the in-game console and run: coi_qol_help

Highlights:
- Extra game speeds beyond vanilla x3
- Vehicle limit controls
- Storage mode tools in GUI and per-storage inspector button
- Terrain tools for instant mine, dump, terrain conversion and trees
- Weather, power, research, unity and worker helpers

Useful commands:
- coi_qol_enable_source_sinks
- coi_qol_unlock_all_research
- coi_qol_instant_build true
- coi_qol_set_game_speed 5
- coi_qol_add_unity 5000
- coi_qol_add_workers 200
- coi_qol_set_storage_mode full
- coi_qol_instant_dump dirt
- coi_qol_fill_storages coal

Notes:
- auto_enable_source_sinks is enabled by default through config.json.
- This mod is designed for Update 4 and uses the current experimental mod API.
- The GUI is an IMGUI overlay and intentionally favors compatibility and fast iteration.
